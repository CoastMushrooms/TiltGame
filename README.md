# TiltGame

## Prerequisites

* Java 17
* Android SDK (API Level 33)
* Git

## Getting Started

### 1. Clone the repository
git clone https://github.com/CoastMushrooms/TiltGame.git
cd TiltGame

### 2. Build the APK
Run the Gradle wrapper to compile the project:
./gradlew assembleDebug

### 3. Locate the APK
After a successful build, the APK file can be found at:
app/build/outputs/apk/debug/app-debug.apk

## Installation
To install the game directly on an Android device:
1. Download the latest APK from the Releases section.
2. Transfer the file to your device.
3. Open the file to install (requires "Install from Unknown Sources" to be enabled).
